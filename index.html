Clarity Test Site

A small static site used to generate session data for the Microsoft Clarity Data Export API. Scroll, click, and navigate between pages to populate the Traffic, Scroll Depth, Engagement Time, and Popular Pages metric blocks.

See pricing

What this page does

Each page on this site loads the Clarity tracking tag and offers enough scrollable content for Clarity to record non-trivial scroll-depth and engagement-time metrics. The nav above links to three sub-pages so the Popular Pages block has more than one URL and referrer to report.

Why a test site

Clarity has no ingestion API. Data is only collected through the in-page JavaScript tag, so a real (or simulated) browser session is required before the export API returns anything useful.
